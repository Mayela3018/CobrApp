# 🤖 CobrApp - Automatización de Pagos Yape/Plin

Sistema automatizado que procesa capturas de pagos Yape/Plin enviadas a Telegram y las registra en Google Sheets, usando **n8n** como orquestador y **Python + easyocr** para el procesamiento OCR.

## 🎯 Problema que resuelve

Academia Fitness Lima recibe mensualmente pagos de más de 200 socios vía Yape/Plin. El administrador pasaba 2-3 horas diarias registrando manualmente las capturas en Excel, con errores y pagos duplicados. Este sistema automatiza todo el proceso.
